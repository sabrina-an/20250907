AI 處理流程教學

▶人臉偵測 (Face Detection)

首先，AI 需要在您的照片(A)中精準地找到人臉的位置，並標記出五官的關鍵點（如眼睛、鼻子、嘴巴和臉部輪廓），這能確保髮型可以被正確地對齊。

// 虛擬程式碼：使用 face-api.js 進行偵測
const detections = await faceapi
  .detectSingleFace(imageA)
  .withFaceLandmarks();
console.log('偵測到臉部關鍵點:', detections.landmarks);

▶髮型分割 (Hair Segmentation)

接著，AI 會分析髮型來源照片(B)，將頭髮的區域從背景中分離出來。這個過程稱為「語意分割」，它能讓 AI 只專注於髮型本身。

// 虛擬程式碼：使用 body-pix 模型
const segmentation = await bodyPix.load();
const { hair } = await segmentation.segmentPerson(imageB);
console.log('成功提取髮型區域');

▶融合與展示 (Image Fusion)

最後一步，是將提取出的髮型(B)套用到您的臉部(A)上。AI 會根據先前偵測到的臉部關鍵點進行縮放、旋轉與對齊，並微調光線與色彩，使其看起來自然融合。

// 虛擬程式碼：使用 Canvas 進行圖像合成
const canvas = document.createElement('canvas');
const ctx = canvas.getContext('2d');
ctx.drawImage(imageA, 0, 0); // 繪製底圖
ctx.drawImage(resizedHair, faceLandmarks.jawline[0].x, faceLandmarks.jawline[0].y); // 疊加髮型
resultImage.src = canvas.toDataURL();
